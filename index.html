URL shortener

How to use this url shortener:

  1. Put the original url after "https://little-url-tobive.herokuapp.com/new/" for example: https://little-url-tobive.herokuapp.com/new/https://example.com
  2. The page will return JSON file containing shortened url
  3. If an error page returned, or the returned JSON file containing null value, please check again the format of the entered url. Acceptable format example: https://www.example.com | https://example.com | http://example.com/folder/blank.html
